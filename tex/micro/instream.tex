\secrel{Входной поток}\label{instream}\secdown

Любое слово имеет доступ к \term{входному потоку} исходного кода, начиная с 
символа после имени этого слова, до конца.\note{в режиме интерпретации}

С помощью этого механизма в \F\ можно вводить собственные модификации
синтаксиса, через определение \term{слов-префиксов}, которые лазят вперёд по коду, и
выполняют с ним нужные действия.
